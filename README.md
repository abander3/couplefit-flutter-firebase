# CoupleFit

CoupleFit is a Flutter + Firebase mobile app designed for couples to track hydration, protein, supplements, workouts, running, weight, and accountability goals together.

## Purpose

This project is being built as a personal learning project and portfolio application to demonstrate mobile app development, Firebase architecture, authentication, Firestore database design, notifications, analytics, UI/UX design, and software project planning.

## Tech Stack

- Flutter
- Firebase Authentication
- Cloud Firestore
- Firebase Cloud Messaging
- Firebase Analytics
- Firebase Storage

## MVP Features

- Daily water tracking
- Daily protein tracking
- Supplement completion tracking
- Workout, walk, and run completion
- Daily weight logging
- Anthony dashboard
- Aspen dashboard
- Shared couple dashboard
- Couple nudges and encouragement messages

## Long-Term Roadmap
